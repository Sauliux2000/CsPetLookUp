1.Have Tampermonkey add-on installed.

2.Press create a new script. 

3.Copy paste the files content to the script.

4.Press ctrl + S to save.

5.That's it.


Script Adds Hyperlink to the majority of pets in Forums,Trades,My stuff for quick navigation to the archives.  
Works with most CS images and also Weebly transparent images
